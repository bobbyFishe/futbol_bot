# Telegram-бот для организации игр в футбол

Этот бот помогает организовывать сборы на футбол (или любые другие командные игры) в групповых чатах Telegram. Он управляет списками игроков, автоматически формирует команды, ведёт статистику игр, штрафы, поддерживает несколько сезонов (баз данных) и гибкие настройки.

## Основные функции

- **Голосование за участие**: игроки пишут `+` (я иду), `-` (я не иду), `+1` (добавляю друга), `-1` (удаляю друга).
- **Автоматическое формирование команд**: при достижении лимита игроков список перемешивается и делится на две команды.
- **Статистика игр**: счёт по голам, очкам, победы, ничьи, история матчей.
- **Штрафы**: система штрафов за неявку/опоздание.
- **Множественные сезоны**: создание, переключение и управление несколькими базами данных (например, «Лето 2025», «Зима 2025»).
- **Гибкие настройки**:
  - Лимит игроков (по умолчанию 14).
  - Названия команд (по умолчанию «Синие» и «Рыжие»).
  - Время голосования (например, только с 9:00 до 21:00 по Москве).
  - Интервал автоматического обнуления списка (в днях).
- **Юмористические сообщения**: бот отвечает случайными фразами (включая сарказм и лёгкий мат).
